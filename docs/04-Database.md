## 4.1 Назначение документа

Настоящий документ определяет структуру базы данных информационной системы FleetOps.

Документ содержит описание используемой системы управления базами данных, структуры таблиц, связей между ними,
первичных и внешних ключей, ограничений целостности, индексов и других элементов, необходимых для хранения данных.

Решения, представленные в настоящем документе, являются основой для реализации моделей Entity Framework Core,
создания миграций базы данных и дальнейшей разработки программного обеспечения.

## 4.2 Выбор СУБД

В качестве системы управления базами данных используется Microsoft SQL Server.

Выбор обусловлен следующими преимуществами:

- высокая производительность;
- тесная интеграция с Entity Framework Core;
- поддержка транзакций;
- широкое распространение в корпоративной разработке;
- развитые средства администрирования.

## 4.3 Структура таблиц базы данных

Каждая последующая версия расширяет структуру базы данных, сохраняя совместимость с предыдущими версиями системы.
